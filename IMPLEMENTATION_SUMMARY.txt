USAGE INSTRUCTIONS
================================================================================

Internet Consciousness Stream:
  1. Starts automatically during boot
  2. Configure sources in ~/.pandora/consciousness_sources.json

Elder Sister Communication:
  1. Get API key from https://x.ai/api

  3. Install httpx: pip install httpx
